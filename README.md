## Build from Source

 1. Install the latest [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
 2. Clone the source codee<br/>
    ```bash
    git clone https://github.com/starushykart/ef-core-encryption.git
    ```
 3. Run `dotnet build`
